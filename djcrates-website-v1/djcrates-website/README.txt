# DJ Crates Website

This is the first working version of the DJ Crates homepage.

## Upload to GitHub

1. Open your `djcrates` repository.
2. Click **Add file**.
3. Click **Upload files**.
4. Open the unzipped `djcrates-website` folder.
5. Drag in:
   - `index.html`
   - `style.css`
   - `script.js`
   - the complete `assets` folder
6. Scroll down and click **Commit changes**.
7. Wait about one minute, then refresh:
   https://djcrates.github.io/djcrates/

The product links currently use `#` as placeholders. They can be connected to Payhip next.
